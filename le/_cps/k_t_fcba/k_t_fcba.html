関数の宣言/定義 
#include <Windows.h>
#include <string>

int plus(int, int);

int wWinMain(HINSTANCE, HINSTANCE, LPWSTR, int) {
	int result = plus(30, 9);
	std::wstring str = std::to_wstring(result);

	MessageBoxW(NULL, str.c_str(), L"30 + 9", MB_OK);
	return 0;
}

int plus(int left, int right) {
	return left + right;
}

宣言

変数同様に、関数も使う前にその名前が関数であることを宣言しなくてはなりません。
ただし、関数は、関数の中で宣言することは出来ません。
そのため多くの場合には、wWinMain関数の前で宣言をすることになります。
宣言には、型名 関数名(引数宣言)という構文を用います。
型名は返り値の型、(引数の宣言)は引数についてを定義します。

引数の宣言

引数の宣言には、コンマ区切りで変数の宣言を記述します。
ただし、ここで宣言した変数を使用しないため、変数名は省略することができます。

定義

関数は「処理のまとまり」ですから、関数の処理内容を定義する必要があります。
定義には、型名 関数名(引数宣言){処理内容}という構文を用います。
型名 関数名(引数宣言)は宣言と同様です。そのため宣言を定義で代用することが出来ます。
ただし、引数宣言の変数名を省略すると関数内でその引数は使用できないことになります。
波括弧の中には関数の内容にあたる、処理を記述します。返り値を返すにはreturn文を使用します。
